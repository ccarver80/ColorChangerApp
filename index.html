Color Changer App

I created a function to pick a random number between 1 and 255

I then created another function for every time the button is clicked to create a "for loop" to loop 3 times and push a random number to the array "num"

This also had a check to see if the array had more than 3 numbers and would delete any extra numbers



Once 3 random numbers got inserted or "pushed" into the array I called the "main" part of the HTML code and did a .style.backgroundColor and added each random number in the array to a RGB using literal notation
